Type: application/x-www-form-urlencoded

Form Parameters
  • username: required (string - pattern: ^\w+@[a-zA-Z_]+?.[a-zA-Z]{2,3}$)

    Unique email address to serve as a username

    Example:

    admin@admin.com
  • password: required (string - minLength: 8 - pattern: ^(?=.*?[A-Z])(?=.*?[a-z])(?=.*?[0-9])(?=.*?[$"?@$!%*#_ +=|~?&()/.,<>`,[\]^"';\\:{}\-]).{8,}$)

    Password must be at least 8 characters long and include at least one of each of 0-9, a-z, A-Z and Symbol (e.g. !

    Example:

    Admin123$$$
  • firstname: required (string)

    User's first name

    Example:

    John
  • lastname: required (string)

    User's last name

    Example:

    Albuquerque
  • middlename: (string)

    User's middle name

    Example:

    Claudio
  • country: required (string)

    User's region/country

    Example:

    USA
  • Any Field Key And Value Here: (string)

    This key will overwrite/update existing value or create a new field.

    Example:

    Name this parameter 'firstName' and value of 'NotJohn'
